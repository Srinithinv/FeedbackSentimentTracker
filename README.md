Feedback & Sentiment Tracker

A full-stack Java-based mini project that collects user feedback, performs sentiment analysis using keyword-based logic, stores data in MySQL, and visualizes results through an interactive analytics dashboard using Chart.js.

FEATURES:
1. User-friendly feedback submission form  
2. Automatic sentiment classification (Positive / Negative / Neutral)  
3. MySQL database storage using JDBC  
4. Real-time analytics dashboard with pie and bar charts  
5. Auto-refreshing chart view  
6. Modern aesthetic UI design for portfolio presentation

TECH STACK:
Backend: Java, JDBC  
Database: MySQL  
Frontend: HTML, CSS, JavaScript  
Visualization: Chart.js
